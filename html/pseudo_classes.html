Pseudo classes

Psuedo classes like :hover and :active

.hover-focus{
padding: 15px;
border: black 5px solid;
font-size: 15px;
}

.hover-focus:hover{
border-color: #0094aa;
}

.hover-focus:focus{
outline: none;
background-color: #aaaaaa;
}

Link pseudo-classes

Links have 4 psuedo classes: :link, :visited, :active, :hover

I am Link
.pseudo-link:link {
color: #0094aa;
font-size: 30px;
}

.pseudo-link:visited {
color: #00FF00;
}

.pseudo-link:hover {
color: #FC440F;
}

.pseudo-link:active {
color: yellow;
}

Table pseudo-classes

By using

Most common pseudo classes

Psuedo-class Description
:active Избира активен линк
:first-child Избира първия елемент от всички
:hover Избира елементи при които мишката е върху тях
:link Избира непосетен линк
:nth-child() nth-child(even) Избира всеки четен елемент, nth-child(odd) всеки нечетен елемент
:visited Избира вече посетен линк
