Text Input





<label for = "first-name"> First name: </label>
<input type = "text" id="first-name"> br br
<label for = "last-name"> Last name: </label>
<input type = "text" id="last-name"> br br
<input type = "submit" value="Submit" class="form-button">

<label> - използва се във формулярите за текст

<input> - използва се за поле за въвеждане, има няколко стойности които са показани по долу

type - определя типът на полето(text-текстово поле, submit-бутон за изпращане на формуляра)

value - използва се ако искаме да сложим предварителен текст в input
